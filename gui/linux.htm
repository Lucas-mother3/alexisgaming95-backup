GNU/Linux

GNU/Linux, more commonly called Linux, is an Unix-like operating system consisting of GNU core utilities and the base Linux kernel. The Linux kernel was first developed by Linus Torvalds in 1992, initially named Freax (Free + Freak + the x suffix). Some popular distributions that uses Linux are Ubuntu, Fedora, Arch, Gentoo, Debian and many others.
